Project Write-Up

This visualization supports interactive exploration of GOES-18 full-disk infrared (Band 13) satellite imagery. The guiding question is how cloud-top brightness temperatures evolve across the Western Hemisphere over several hours. We focus on large-scale spatial patterns rather than fine local detail. This scope keeps the visualization focused and feasible within the two-week window.

The raw satellite data were downsampled to a 50×50 grid to balance clarity and performance. We aggregated brightness temperature by spatial cell and time step. We export three static files for the web app: grid geometry, time metadata, and cell-by-time values. This transformation preserves structure while keeping file sizes small enough for fast loading.

Brightness temperature is encoded with a sequential inferno color scale in the main heatmap. Position is used for all quantitative comparisons in the linked time-series view. Time is shown both through animation and a scrubber slider. Hover reveals details on demand, and clicking a cell pins a reference series for comparison.

We overlay coastlines using a D3 orthographic projection as an intentional approximation. GOES data use a true geostationary projection that is mathematically complex to replicate in the browser. We favored a visually aligned approximation for elegance, performance, and development time. The approximation improves geographic readability without introducing perceptual confusion for the intended tasks.

Development was split between Python data processing and D3 front-end implementation. Approximately equal time was spent on data access, transformation, and interactive design. The most time-intensive challenges were GOES file access and projection alignment decisions.

A natural future direction is an automated pipeline that fetches and processes GOES data for a user-specified date range. This would enable exploratory analysis across days or seasonal events. Anticipated challenges include large data volumes, API rate limits, and long preprocessing times.
